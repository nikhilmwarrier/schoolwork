Bold and Italics

<b>Bold</b> & <i>Italicized</i>
		<!--Or [recommended]  -->
<strong>Bold</strong> & <em>Italicized</em>
Result:
Bold & Italicized
Note: It is recommended to stop using presentational tags like <b> and <i> in favor of their more semantic counterparts <strong> and <em>.

The <p> tag:

<p> Lorem ipsum, dolor sit amet consectetur adipisicing elit. Aliquid, dolores? </p>
Result:
Lorem ipsum, dolor sit amet consectetur adipisicing elit. Aliquid, dolores?

<font> tag

Warning: Deprecated. Avoid using presentational HTML tags.
Just use font and color CSS properties instead. 
<font face="sans-serif" color="green"> Hello I'm deprecated! Whee! </font>
Result:
Hello I'm deprecated! Whee!
